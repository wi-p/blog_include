A programação é a área da informática que trabalha com o desenvolvimento de softwares, como aplicativos, banco de dados, front e back. É uma área bem diversificada, possuindo ramos que vão do desenvolvimento mobile até redes neurais.

A seguir temos as principais áreas da programação:

  • Desenvolvimento mobile;
  • Desenvolvimento desktop;
  • Machine learning;
  • Inteligência artifical (IA);
  • Redes neurais;
  • Desenvolvimento web.

Existem muitas outras, mas essas são as principais.
